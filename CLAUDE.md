# 지시사항

## 언어
항상 한국어로 대답한다.

## 브랜드
- 브랜드명: OFFEGO
- 카테고리: 무니코틴 흡입기
- 슬로건: Drop self they built.

## 카피라이팅
카피는 짧고 단정적으로 작성한다.

## 시장조사
시장조사 결과는 인사이트 중심으로 요약한다.

## 결과물 출력 방식
- 결과물은 항상 HTML 파일로 만들고 브라우저로 띄워줘
- 텍스트로만 정리하지 말고 시각적으로 보여줘
- 파일 저장 시 항상 절대경로 사용: C:\Users\zang0\Desktop\my-site\
- 애매한 지시도 최대한 의도를 파악해서 알아서 진행
- 중간에 확인 질문 줄이고 판단해서 실행
- 작업 완료 후 결과를 브라우저로 바로 열기:
  powershell -command "Start-Process 'C:\Program Files\Google\Chrome\Application\chrome.exe' -ArgumentList 'file:///C:/Users/zang0/Desktop/my-site/파일명.html'"
